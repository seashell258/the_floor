| 項目 | 路徑 |

| 圖片目錄 | `/app/台灣行道樹＆園藝植物/` |
| 圖片命名 | `1.jpg`–`36.jpg`（混合 jpg/jpeg/avif/JPG 副檔名） |

## 頁面佈局

- 尺寸：A4 直向（595 × 842 pt）
- 外邊距：10 pt（四周）
- 圖片區：頁面上方，最大高度 660 pt（約 80%），等比縮放，水平置中
- 文字區：圖片下方剩餘空間，植物名稱垂直水平置中，字體 28 pt

```
┌──────────────────────┐
│  10pt padding        │
│  ┌────────────────┐  │
│  │   圖片（等比）  │  │  最高 660pt
│  └────────────────┘  │
│  ────────────────    │
│    植物名稱 28pt     │  剩餘空間置中
│  10pt padding        │
└──────────────────────┘
```

## 腳本設計

**檔案：** `/app/generate_pdf.py`

**依賴：**
- `reportlab` — PDF 生成
- `Pillow` — 圖片讀取與格式統一轉換（處理 avif、大寫副檔名等）

**排序：** 以數字排序（1, 2, 3 … 36），避免字母排序的 1, 10, 11 問題。

**圖片查找策略：** 對每個編號 `n`，嘗試副檔名 `jpg`、`jpeg`、`JPG`、`JPEG`、`avif`、`png`，取第一個存在的檔案。

**中文字體：** 執行時自動搜尋系統 TTF 字體，優先選擇含「CJK」「Noto」「黑體」關鍵字的字體；若找不到則報錯並提示使用者手動指定路徑。

**錯誤處理：** 找不到某頁圖片時，該頁顯示紅色錯誤訊息並繼續生成其餘頁。

## 執行方式

```bash
pip install reportlab pillow
python generate_pdf.py
```

輸出：`台灣行道樹＆園藝植物.pdf`（36 頁）
