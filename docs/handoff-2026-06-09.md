**Vote bar structure — approved design:**

Two sub-columns side by side inside `.vote-bar`:
- Each sub-column is ~36px wide, 8px gap between them
- Structure per sub-column (top to bottom):
  1. Vote count number (e.g. `3`)
  2. Vertical track (flex: 1, `var(--bg-surface)` background)
  3. Fill div inside track, grows from bottom, height = vote percentage
  4. Player name in `writing-mode: vertical-rl` at the base

- **P1 fill color:** `var(--glow)` (cyan) + `box-shadow: 0 0 8px var(--glow-30)`
- **P2 fill color:** `var(--warn)` (amber) + `box-shadow: 0 0 8px rgba(245,158,11,0.3)`
- **Fill animation:** `transition: height 0.4s ease`
- **Height formula:** `votes1 / (votes1 + votes2) * 100` → clamped to `[0, 100]`, 0 when total=0

**Script change:** add `getVotePercent(playerNum: 1 | 2): number` method to `<script setup>`:
```ts
function getVotePercent(playerNum: 1 | 2): number {
  const v1 = voteResults.value?.votes1 ?? 0
  const v2 = voteResults.value?.votes2 ?? 0
  const total = v1 + v2
  if (total === 0) return 0
  return playerNum === 1 ? (v1 / total) * 100 : (v2 / total) * 100
}
```

**answer-panel change:** Remove `.answer-header` div entirely from template (vote counts moved to vote-bar). Only `.answer-body` remains inside `.answer-panel`.

**File to change:** `src/components/Dashboard/BattleSection.vue` only.
